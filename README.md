## Ejecutar Localmente

**Requisitos:** Node.js

1.  **Instalar dependencias:**
    ```bash
    npm install
    ```
2.  **Configurar la clave de API:**
    Crea un archivo `.env.local` en la raíz del proyecto y añade tu clave de API de Gemini:
    ```
    GEMINI_API_KEY=TU_API_KEY_AQUI
    ```
3.  **Ejecutar la aplicación:**
    ```bash
    npm run dev
    ```
